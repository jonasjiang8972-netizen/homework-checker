# 贡献者公约

## 我们的承诺

参与本项目，我们承诺营造一个开放、友好的环境。我们致力于：

- **尊重和包容**：尊重不同的观点、经验和身份，创造无骚扰的社区环境。
- **专业性**：以专业态度对待彼此，关注共同目标。
- **建设性反馈**：提供有帮助、有建设性的反馈，避免人身攻击。
- **共同目标**：关注对项目最有利的事情，服务学习者社区。

## 我们的标准

有助于社区的行为：

- 使用友好、包容的语言
- 尊重不同的观点和经验
- 接受建设性批评
- 关注社区共同利益
- 对其他参与者表现出同理心

不可接受的行为：

- 使用性化的语言或图像，任何形式的不当身体接触
- 人身攻击、嘲讽、羞辱性评论
- 公开或私下骚扰
- 未经授权的发布信息、隐私泄露
- 其他不道德或不专业的行为

## 执行责任

项目维护者有责任执行可接受的行为准则，并有权删除、编辑或拒绝评论、提交、代码、编辑、issue 等其他贡献，以维护项目环境。

## 执行

发现违反行为准则的实例，可向项目维护者报告。所有投诉将审查并及时、公平地处理。

项目维护者若违反行为准则，将面临相应后果，可能包括从项目中移除。

## 适用范围

本公约适用于所有项目空间和公共场合，当个人代表项目或其社区时。
