### 2.2 API Key 加密管理

| 模块 | 状态 |
|------|------|
| AES-256-GCM 加密工具 `lib/encryption.ts` | ✅ 已完成 |
| `user_settings` 表设计（加密 Key + 偏好） | ✅ 已完成 |
| `POST /api/user/key` 加密存储用户 Key | ✅ 已完成 |
| `GET /api/user/key` 返回脱敏状态（无明文） | ✅ 已完成 |
| `DELETE /api/user/key` 删除用户 Key | ✅ 已完成 |

**加密方案**：
- 使用 Node.js 内置 `crypto` 模块（零额外依赖）
- AES-256-GCM 加密，随机 IV，附加 authTag
- 主密钥来源于环境变量 `API_KEY_ENCRYPTION_SECRET`（fallback 到 `NEXTAUTH_SECRET`）
- 数据库中仅存储加密后的密文，永不明文存 Key
