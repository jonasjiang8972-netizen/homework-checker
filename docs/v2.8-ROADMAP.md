## 1. 升级目标

v2.6~v2.7 完成了安全加固和依赖升级。v2.8 聚焦**产品定位调整**——从"家长批改工具"转向"初中生学习助手"，核心改动：

- **两段式批改**：先引导、孩子点了还是不会才给答案，防止变成抄答案工具
- **隐私模式**：学生模式仅展示趋势汇总，不暴露单题细节，13 岁孩子才愿意主动使用
- **安全清理**：移除泄露的部署环境信息

**交互流程**：
```
批改完成
  │
  ▼
  ✅/❌ 判断 + 订正类型 + 知识点
  │
  ├─ 💡 想一想："检查一下去括号时符号有没有变对？"
  │
  ├─ [还是不太懂，给我看看答案 →]  ← 用户点击后才展开
  │     ├─ 🔍 你的订正
  │     ├─ ✏️ 正确答案
  │     └─ 💡 为什么错了
  │
  └─ [记下来 📖]
```

**Prompt 改造要点**：
- 新增 `guidance` 字段：引导性提示，不给答案，帮孩子自己发现错误
- 保持 `correct_solution` 和 `analysis` 字段（点击答案后展示）
- 全对时 `guidance` 为空字符串

## 3. 产品定位调整

### 3.1 从"家长批改工具"到"初中生学习助手"

| 维度 | v2.7 及之前 | v2.8 |
|------|------------|------|
| 目标用户 | 家长（拍照者） | **初中生本人（使用者）** |
| 批改策略 | 直接给完整答案 | **先引导启发，主动请求才给答案** |
| 数据可见性 | 全部公开 | **学生模式仅见汇总，家长可见详情** |
| 语气 | 面向小学生（"真棒！"） | **面向初中生（简洁直接，不幼稚）** |
| Prompt 示例 | "两位数进位加法" | **"一元一次方程""去括号变号"** |

### 3.2 后续规划（v2.9+）

- **校园场景适配**：初中物理/化学学科支持
- **自定义引导强度**：学生可设置"直接给答案"还是"多步引导"
- **学习习惯分析**：基于使用时长、时间段、学科分布给出学习建议
