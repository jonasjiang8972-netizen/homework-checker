## 2. 功能拆解

### 2.1 批改结果 Markdown 渲染（公式/列表美化）

| 模块 | 状态 |
|------|------|
| 安装 KaTeX 公式渲染依赖 | ✅ 已完成 |
| 创建 `MarkdownRenderer` 共享组件 | ✅ 已完成 |
| 首页批改结果卡片接入 Markdown 渲染 | ✅ 已完成 |
| 错题本分析区域增强公式支持 | ✅ 已完成 |

后端 prompt 中已包含 Markdown 格式的解题过程（如 `## 第一步`、`$27+38=65$` 等），前端需要正确渲染。

**实现方案**：
- 使用 `marked`（已有依赖）+ `katex` 渲染行内/块级公式
- 支持标准 Markdown：标题、列表、表格、代码块、加粗、公式
- 组件化，首页和历史页共用

**验收标准**：
- `正确解答` 字段中的 Markdown 列表、标题正确渲染
- `$...$` 和 `$$...$$` 公式显示为数学符号
- 代码块有语法高亮背景
