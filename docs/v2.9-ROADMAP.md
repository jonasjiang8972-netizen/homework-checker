| 模块 | 状态 |

| `handleGenerate` 存 `questions_json` 加 `JSON.stringify` | ✅ 已完成 |
| `handleSubmit` 读 `questions_json` 加 `parseQuestionsJson()` | ✅ 已完成 |

| `GET /api/quiz` 返回前自动解析 JSON 字段 | ✅ 已完成 |
| 兼容旧数据（Array/String 双类型处理、parse 失败降级空数组） | ✅ 已完成 |

**根因**：
```
supabase mock → insertExec(vals) → sql.js db.run(sql, vals)
→ Array.toString() → "[object Object]" 写入 SQLite
→ 读取时 record.questions_json 为乱码字符串
→ questions.map() 报错 → catch → "AI 服务暂时不可用"
```
