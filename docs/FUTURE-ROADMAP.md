# 未来优化方向（v2.9+）

> v2.8 已完成（两段式批改 + 隐私模式），以下为 v2.9 之后的长期规划。

---

## 🎯 v2.9+ 规划

### 1. 校园场景适配
- 初中物理/化学学科支持（公式、化学方程式渲染）
- 英语作文批改（语法纠错）
- 自定义知识图谱

### 2. 引导策略进阶
- 多级引导强度（学生自定义：直接给答案 / 给提示 / 分步骤引导）
- 错题关联引导（"你上次在同一知识点错的是这道题"）
- AI 根据历史错题动态调整引导方式

### 3. 数据洞察增强
- 学习趋势图表（周/月维度）
- 薄弱知识点变化追踪
- 基于使用时段的效率分析

### 4. 生态与集成
- 错题导出（PDF/图片）
- 多端数据同步（手机/平板/电脑）
- 家长通知（周报邮件，仅含汇总信息）

### 5. 高阶交互
- 语音输入答案
- 公式手写识别
